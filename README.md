# Supplementary Information

for the article **ChemVLMate: a multimodal, dual-arm, seeing robotic chemist for adaptive synthesis** 

## Directories

* `codes_for_paper` — Source code developed for ChemVLMate, including the RGB image acquisition program, VLM Perception Module, LLM Reasoning Module, and Robotic Manipulation Module.
* `data_for_paper` — Experimental data collected from Task 4.
